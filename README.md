# Quiz-Game-Module
A Quiz Module which has some interesting quizzes for users to solve under time constraint of 15 Min. This is fully made of HTML5(DOM Manipulation), CSS, JavaScript, and Bootstrap. 
# Technology
* HTML5
* CSS
* JavaScript
* Bootstrap 4
